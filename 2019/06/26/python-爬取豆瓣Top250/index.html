爬取豆瓣Top250

语言：python
使用库：requests(http请求数据), json(字典与json转换), xlwt(json与excel),lxml(解析xml，xpath)
爬取内容：豆瓣电影，图书，音乐Top250

主要方法步骤

  • 使用requests请求网页url，然后使用lxml把网页html解析解析下来
  • 获取信息区块列表
  • 分别爬取区块想要的信息存入字典
  • 写入文件保存

值得注意点

  • 在使用pip安装的时候,由于是国外资源,下载速度很慢,可以使用国内镜像, 在cdm中输入
